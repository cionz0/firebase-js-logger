# firebase-js-logger


# Installation
```
npm install --save git+https://github.com/cionz0/firebase-js-logger.git  
```

# Usage
In your `index.js`:
```
const logger = require("@cionzo/firebase-js-logger").createLogger(__dirname);
```

In any other module:
```
const logger = require("@cionzo/firebase-js-logger")
```
